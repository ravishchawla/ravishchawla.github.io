Page Rank

Wrote Map reduce algorithms for Big data processing for Page Rank in Hadoop, to process the Wikipedia page-links dataset and Twitter-Memetracket dataset. The page rank algorithm ranked websites based on how many other websites link to it, by treating hyperlinks as edges and websites as nodes. The data collected from the Map Reduce implementation was then evaluated against an Apache Giraph implementation to compare memory and speed processing of the algorithms.

Presentaion Link: Result and analysis of Page rank and comparison of implementations.

BitTorrent Application

Developed a BitTorrent application in C#, that can download and upload torrent files from secure trackers. The application used a Bencoding library to read torrent file data, and connected to peers using a persistent TCP connection. File download took place with parallel unchoked connections to multiple peers, and handled for packet corruption, and uploaded file chunks as they were received.

ArangoDB

Researched a new NoSQL Technology called ArangoDB, which provides Graph data storage in terms of nodes and edges, as well as documents. Compared it to other databases, such as CouchDB and MongoDB, analysing what features they provide, and how performance compares between each for large amounts of data.

Research Paper Link: ArangoDB - A new NoSQL Database System
